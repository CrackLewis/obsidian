
DOM是W3C标准的一部分。W3C DOM可细分为：Core DOM、XML DOM、HTML DOM。

当网页被加载时，浏览器会创建页面的文档对象模型（Document Object Model）。 **通过 HTML DOM，JavaScript 能够访问和改变 HTML 文档的所有元素。**

HTML DOM是HTML的标准对象模型和编程接口。它定义了：
- 作为对象的HTML元素
- 所有HTML元素的属性
- 访问所有HTML元素的方法
- 所有HTML元素的事件
