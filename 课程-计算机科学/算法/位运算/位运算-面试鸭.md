
## 判断一个数是否为2的幂

`bool is_pow2 = ((x > 0) && (x & (x - 1)) == 0)`

## 找出不低于一个整数的最小的2的幂

```cpp
int getNextPowerOfTwo(int n) {
    if (n == 0) return 1;
    n--;
    n |= n >> 1;
    n |= n >> 2;
    n |= n >> 4;
    n |= n >> 8;
    n |= n >> 16;
    return n + 1;
}
```

## 统计一个数中1的个数

```cpp
int countOnes(int n) {
    int count = 0;
    while (n != 0) {
        n &= (n - 1);
        count++;
    }
    return count;
}
```
