 struct linked_node_t {
	 linked_node_t *next;
	 int val;
 };

## 链表中点

```cpp
linked_node_t *slow = head, *fast = head, *prev;
while (fast && fast->next) {
	prev = slow;
	slow = slow->next;
	fast = fast->next->next;
}
// prev：中点前驱节点（方便断链做归并排序）
// slow：中点
```
