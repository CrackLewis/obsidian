## 搭建verilator仿真环境

verilator是一种开源仿真器，可实现将Verilog转换为C/C++代码。

由于实验强烈要求5.008版本，所以从指定tag克隆：

```bash
$ git clone -b v5.008 git@github.com:verilator/verilator.git
```
